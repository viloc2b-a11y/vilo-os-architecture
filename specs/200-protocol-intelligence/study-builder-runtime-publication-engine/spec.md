\# Study Builder and Runtime Publication Engine



Status: Draft



\---



\# Objective



Define the architecture responsible for transforming protocol definitions, visit structures, procedures, source requirements, workflow logic, and operational rules into executable runtime study configurations.



\---



\# Core Principle



Study setup is not document configuration.



Study setup is runtime publication.



\---



\# Responsibilities



The engine must support:



\- protocol version definition

\- visit matrix construction

\- procedure mapping

\- source template linkage

\- workflow rule definition

\- operational rule linkage

\- publication versioning

\- amendment publication

\- runtime freezing

\- executable runtime generation



\---



\# Publication Chain



Protocol Definition

→ Visit Matrix

→ Procedure Graph

→ Source Linkage

→ Workflow Rules

→ Runtime Validation

→ Runtime Publication

→ Immutable Runtime Version

\# Runtime Publication

\- immutable runtime configuration

\- source package bindings

\---



\# Publication Validation



The system must validate:



\- orphan procedures

\- missing source linkage

\- missing financial mappings

\- invalid dependencies

\- unresolved workflow conflicts

\- invalid visit windows

\- missing signatures

\- amendment conflicts



\---



\# Amendment Publication



Amendments must support:



\- delta computation

\- runtime versioning

\- longitudinal continuity

\- prior-runtime preservation

\- retraining triggers

\- financial delta generation



\---



\# Runtime Freezing



Published runtime versions must:



\- remain immutable

\- preserve lineage

\- preserve historical execution context



\---



\# UX Requirements



Study builders must experience:



\- protocol-aware setup

\- operational visibility

\- dependency clarity

\- publication confidence

\- amendment explainability



without manually rebuilding runtime logic.



\---



\# MVP Scope



MVP must support:



\- visit matrix builder

\- procedure mapping

\- source linkage

\- runtime publication

\- immutable publication versions

\- amendment placeholders

\- publication validation



\---



\# Deferred Scope



Deferred:



\- autonomous protocol parsing publication

\- sponsor self-service setup

\- visual enterprise workflow marketplace

\- external standards auto-mapping



\---



\# Explicit Non-Goals



The builder is NOT:



\- generic form builder

\- spreadsheet replacement only

\- sponsor configuration portal

\- static study setup wizard



\---



\# Final Rule



If a published study cannot execute operationally without engineering intervention, the publication architecture has failed.
